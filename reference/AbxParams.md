# Antibiotic Parameters

Antibiotic Parameters

## Usage

``` r
AbxParams(onoff = FALSE, delay = 0, life = 1)
```

## Arguments

- onoff:

  If Anti-biotic are used or not.

- delay:

  The delay in using antibiotics.

- life:

  The life of antibiotics.

## Examples

``` r
AbxParams()
#> $onoff
#> [1] FALSE
#> 
#> $delay
#> [1] 0
#> 
#> $life
#> [1] 1
#> 
```
